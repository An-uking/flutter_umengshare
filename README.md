# flutter_umengshare

友盟分享插件 for Flutter

## 说明：
目前当前库添加了微博、qq、微信这个几个库
因为要配置很多参数，我没有另写加载函数，也方便你们自己扩展其他分享和登陆的第三方库，如 facebook、twitter等可自己到友盟去下载相应的包加入到其中.
